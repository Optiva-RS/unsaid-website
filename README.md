# Unsaid Website

Public marketing, privacy, terms, and support website for Unsaid.

## Website files

- `index.html` - Main marketing page
- `privacy.html` - Privacy Policy
- `terms.html` - Terms of Service
- `support.html` - Customer support information
- `styles.css` - Website styling
- `assets/` - Logo and website imagery

## Hosting with IONOS

This is a static website and does not require a database, Node.js, or a build command.

When connecting the repository to IONOS:

- Repository: `Optiva-RS/unsaid-website`
- Branch: `main`
- Build command: leave blank
- Publish or output directory: `/` (repository root)
- Entry file: `index.html`

The public website intentionally does not link to the private Base44 application URL.
